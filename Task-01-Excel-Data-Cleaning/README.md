# Task 01 – Excel Data Cleaning & Formatting

## Internship
**Role:** Data Analyst Intern  
**Organization:** Elevate Labs  
**Duration:** Jan 2026 – Apr 2026  

## Objective
To clean and standardize a real-world HR Analytics dataset using Microsoft Excel, ensuring data quality, consistency, and readiness for analysis.

## Dataset
HR Analytics Dataset

## Tools Used
- Microsoft Excel

## Key Steps Performed
- Preserved raw data and created backup before modifications
- Identified and validated missing values
- Detected and safely removed confirmed duplicate records
- Standardized categorical text fields using TRIM, PROPER, and UPPER
- Validated numeric data types and formats
- Created a cleaned, analysis-ready dataset
- Added data quality documentation

## Deliverables
- Raw_Data.xlsx  
- Cleaned_dataset.xlsx  
- cleaned_dataset.csv  

## Outcome
A clean, structured dataset ready for further analysis and reporting, following professional data-cleaning practices.
