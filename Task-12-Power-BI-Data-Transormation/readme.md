# Data Analyst Internship – Task 12

## Task Title
Power BI – Data Transformation using Power Query

## Dataset Used
HR Analytics Dataset (CSV)

## Tools Used
- Power BI Desktop
- Power Query Editor

## Task Objective
To clean, transform, and prepare HR data using Power Query before creating visuals in Power BI.

## Data Transformation Steps
1. Imported HR_Analytics.csv into Power BI.
2. Opened Power Query Editor.
3. Promoted first row as headers.
4. Removed unnecessary columns (EmployeeCount, StandardHours, Over18, EmployeeNumber).
5. Renamed columns for clarity (SalarySlab → Salary Band).
6. Checked and corrected data types.
7. Created conditional columns:
   - AgeGroup
   - Salary Band
8. Applied changes and loaded the data model.

## Visualizations Created
1. Attrition by Department (Stacked Bar Chart)
2. Employees by Age Group (Column Chart)
3. Salary Band Distribution (Pie Chart)
