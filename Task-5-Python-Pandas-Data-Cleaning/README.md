# Task 5 – Python Basics: Data Cleaning Using Pandas

## Internship
Elevate Labs – Data Analyst Internship

## Objective
The objective of this task is to perform data cleaning and preprocessing using
Python and Pandas.

## Dataset Used
Medical Insurance Cost Dataset (insurance.csv)

## Tools & Libraries
- Google Colab
- Python
- Pandas
- NumPy

## Steps Performed
- Loaded the dataset into a Pandas DataFrame
- Explored dataset structure using info() and describe()
- Checked and handled missing values
- Identified and removed duplicate records
- Verified and converted data types
- Performed basic feature engineering
- Exported the cleaned dataset

## Outcome
This task improved hands-on experience in data preprocessing and data cleaning
using Pandas and demonstrated how Python simplifies data handling compared to
manual tools.
