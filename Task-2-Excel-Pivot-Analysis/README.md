# Task 2 – Excel Pivot Table Analysis

## Objective
To analyze e-commerce sales data using Excel Pivot Tables and generate meaningful business insights.

## Dataset
Ecommerce Sales Prediction Dataset

## Tools Used
- Microsoft Excel

## Analysis Performed
- Created Total_Sales metric (Price × Units_Sold)
- Pivot Table: Total Sales by Product Category
- Pivot Table: Total Sales by Customer Segment
- Interactive slicers for dynamic filtering
- Pivot chart for visual comparison

## Key Insights
- Sports is the top-performing product category.
- Home Decor has the lowest total sales.
- Occasional customers contribute the highest share of sales.
- Sales are well distributed across major categories.
- Customer segmentation significantly impacts sales performance.

## Outcome
This task demonstrates data summarization, business analysis, and analytical storytelling using Excel.
